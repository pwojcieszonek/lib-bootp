# Lib::BOOTP

BOOT Library written in ruby. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'lib-bootp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lib-bootp
